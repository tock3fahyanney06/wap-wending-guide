# wap-wending-guide

这是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为页面资源的结构化存放与版本管理用途。

## 项目简介

本仓库旨在整理和保存若干独立 HTML 页面，便于长期查阅、分发与维护。每个页面均为自包含的静态文件，不依赖外部特定服务或数据库。仓库内容可视为一份可离线浏览的页面集合。

## 目录说明

```
.
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面依赖的资源文件（样式、脚本、图片等）
├── index.html      # 仓库首页或导航页（可选）
└── README.md       # 本说明文件
```

- `pages/`：按功能或主题划分的页面目录，每个子目录通常包含一个或多个 HTML 文件。
- `assets/`：存放页面共用的 CSS、JavaScript、字体、图像等静态资源。
- `index.html`：可选的入口页面，用于快速浏览仓库内所有页面链接。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开使用。
- 页面之间可能存在相互引用，建议保持目录结构完整。
- 部分页面可能依赖 `assets/` 目录下的资源文件，请勿随意移动或删除。
- 归档内容不包含任何动态后端代码，无需服务器环境即可运行。

## 维护说明

- 欢迎提交 Issue 报告页面问题或提出改进建议。
- 如有新增页面，请遵循现有目录结构，并确保资源引用路径正确。
- 修改已有页面时，建议在提交信息中简要说明变更内容。
- 本仓库为静态归档性质，原则上不提供实时更新或技术支持。
